# Product Review Sentiment Analysis

## Overview
This project is a **sentiment analysis web application** that classifies product reviews as **positive, negative, or neutral** using a machine learning model. Users can input their review text via a web interface, and the app predicts the sentiment in real-time.

## Tech Stack
- **Backend:** Python, Flask  
- **Machine Learning:** Scikit-Learn, NLTK (Natural Language Toolkit)  
- **Web Interface:** HTML, CSS, Flask Templates  
- **Model Persistence:** Pickle  

## Features
- Preprocesses text data (lowercasing, removing special characters, stemming)  
- Predicts sentiment using a trained ML model  
- Supports multiple ML models (e.g., Logistic Regression, SVM, Random Forest)  
- Provides a simple and responsive web interface for user input
